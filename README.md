# Magistus

Aplicação React + Vite para transformar uma planilha em fichas.

## Como usar

1. Execute `npm install`.
2. Inicie o projeto com `npm run dev`.
3. Faça upload de um arquivo Excel ou CSV.
4. Cada linha da planilha vira uma ficha com os valores das colunas.

## Scripts

- `npm run dev` — inicia o ambiente local.
- `npm run build` — gera a versão de produção.
- `npm run test` — executa os testes unitários.
